# Feature Access Matrix

## Overview

The Feature Access Matrix defines which features are available under each subscription plan.

It serves as the single source of truth for:

- Frontend UI
- Backend Authorization
- Pricing Page
- API Validation
- Feature Flags
- Product Planning

Every premium feature must be validated by the backend before execution.

---

# Subscription Plans

| Plan | Description |
|-------|-------------|
| Free | Basic productivity features with limited AI usage |
| Pro | Full AI-powered personal productivity workspace |
| Team | Collaboration, administration, and team productivity |
