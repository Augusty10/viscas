# Technology Stack

## Overview

Viscas is built using a modern full-stack technology stack that emphasizes scalability, security, maintainability, and performance. Each technology has been carefully selected based on project requirements, industry adoption, and long-term sustainability.

The architecture follows a modern SaaS development approach, separating the frontend, backend, database, AI services, and deployment infrastructure into independent layers.

---

# Technology Stack Overview

| Layer | Technology |
|--------|------------|
| Frontend | Next.js |
